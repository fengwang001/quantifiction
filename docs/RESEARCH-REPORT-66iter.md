# 研究报告：把 $100 高频量化榨到市场效率墙

> 66 轮评估 · 1825 笔真实行情纸面交易 · 21 个策略 · 3 个标的(ETH/BTC/SOL) · 零真金风险
> 可视化版本：[`RESEARCH-REPORT-66iter.html`](RESEARCH-REPORT-66iter.html)（含费用墙数轴图、策略全景条形图，支持深/浅色）

## 核心结论

**策略有真实的正 edge——但它恰好卡在 maker 费与 taker 费之间。** 微观结构信号的可预测性已被市场充分定价，剩下的毛利薄到无法扣费后稳定盈利。这不是策略缺陷，是**市场效率墙**。

整个实验在**影子模式**下进行：策略按欧易真实行情成交、扣真实手续费记账，但不下真单——盈亏结论可信，风险为零。

## 决定性发现：费用墙

把最优策略（均值回归·优化出场）的毛利除到每笔：

```
edge = +0.041 / 笔        ← 真实、持续的正 edge
maker 费 = 0.040 / 笔
taker 费 = 0.100 / 笔
──────────────────────────────
maker 净 = 0.041 − 0.040 = +0.001 ≈ 0   ← 安全边际蒸发
taker 净 = 0.041 − 0.100 = −0.059         ← 清晰为负
```

Edge 随样本单调收窄：`+0.102 → +0.068 → +0.041`。第 61 轮那个净值转正的"突破"，被更大样本证明是 regime 运气。真实 edge 约 +0.04/笔——**刚好覆盖不住 maker 费，更别提 taker。**

## 进化证据链：66 轮里学到的六件事

每一条都由真实成交数据逼出来，老策略与数据全程保留作对照（宪法 V）。

1. **分钟级高频必亏** — 止盈从未触发、信号无预测力，净亏 = 手续费（1215 笔）
2. **换到小时级波段** — 单笔毛利变成手续费的 3–8 倍（TP 0.8–1.5%，持仓 2–4h）
3. **固定止盈收集回吐 → 追踪止损** — 浮盈回吐过半，trailing 锁利（早期 17/17 全胜）
4. **固定 gap 不缩放 → 比例追踪** — 锁定峰值 60%，随行情自动缩放（同笔 +0.070 vs 固定宽 −0.110）
5. **一笔止损抹掉四笔盈利 → 紧止损** — 34 笔止损 74% 从没顺过，收紧削亏，盈亏比 0.51→0.71
6. **最佳出场≠通用** — 紧止损利于动量却误伤均值回归；出场机制与信号相互作用，别被小样本带偏

## Pivot：微观结构榨干后转资金费 carry

微观结构触墙后，转向不被 HFT 竞争掉的结构性 edge——永续每 8h 的资金费。

- **为什么**：资金费是持仓补偿，结构性存在。市场中性建模（perp + spot 对冲），价格 PnL 抵消，只收资金费。
- **当前现实**：费率过低（ETH 年化 0.7%、BTC 4.4%），且中性对冲 spot 腿费用高。carry 引擎正确空仓等待，不强凑交易。
- carry 只在费率走高时才明显盈利（牛市资金费常冲到 0.05–0.1%/8h ≈ 年化 50–100%）。

## 这份报告真正证明了什么

把一堆注定亏损的高频信号，经过 66 轮数据驱动的进化，打磨到**逼近盈亏平衡、毛利转正**，并精确定位到**费用/市场效率**是唯一剩余障碍——这本身就是扎实的科学成果。

它诚实地回答了"$100 高频量化能否稳定盈利"：**在当前平静、低波动、低资金费的 regime 下，零售费率会吃掉那点微弱的 edge。**

**三条真正能改变局面的路**：
1. 等一个高波动 / 高资金费的 regime
2. 换更低费率的执行（**maker 限价挂单** / VIP 返佣）
3. 接受"当前无 edge"并停手

**一条已被数据钉死的工程铁律**：实盘必须用限价挂单（maker），绝不能用市价单（taker）——否则再好的策略，都会被 0.10% 的 taker 费吃平。
