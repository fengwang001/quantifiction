核心结论

策略有真实的正 edge——但它恰好卡在 maker 费与 taker 费之间。微观结构信号的可预测性已被市场充分定价,剩下的毛利薄到无法扣费后稳定盈利。这不是策略缺陷,是市场效率墙。

Pivot · 转向结构性 edge

微观结构榨干后,转资金费 carry

既然微观结构信号触墙,唯一出路是找一个不被 HFT 竞争掉的结构性 edge——永续每 8h 的资金费。

为什么是 carry

资金费是持仓补偿,结构性存在。市场中性建模(perp + spot 对冲),价格 PnL 抵消,只收资金费——不赌方向。

当前的诚实现实

费率过低(ETH 年化 0.7%、BTC 4.2%),且中性对冲的 spot 腿费用高。carry 引擎正确地空仓等待,不强凑交易。

carry 只在费率走高时才明显盈利(牛市资金费常冲到 0.05–0.1%/8h ≈ 年化 50–100%)。引擎已就位,等一个更好的 regime。

不能稳定盈利,是一个有价值的结论

把一堆注定亏损的高频信号,经过 66 轮数据驱动的进化,打磨到逼近盈亏平衡、毛利转正,并精确定位到费用/市场效率是唯一剩余障碍——这本身就是扎实的科学成果。它诚实地回答了"$100 高频量化能否稳定盈利":在当前平静、低波动、低资金费的 regime 下,零售费率会吃掉那点微弱的 edge。

真正能改变局面的三条路:①等一个高波动 / 高资金费的 regime;②换更低费率的执行(maker 限价挂单 / VIP 返佣);③接受"当前无 edge"并停手。继续硬调参数只会重蹈手续费血亏的覆辙。

而最重要的一条工程铁律,已经被数据钉死:实盘必须用限价挂单(maker),绝不能用市价单(taker)——否则再好的策略,都会被 0.10% 的 taker 费吃平。
